littleWorld
===========

用于参加七牛比赛

启动方式，依次双击 1-pomelo_start.bat，2-open-crossdomain.bat，3-web-start.bat 这三个文件就可以了，然后使用浏览器登陆3001端口，点击“打开flash”就可以看到flash了。
